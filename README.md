# Gamer Lobby Personal Website

A dark, minimal, gamer-themed personal website that feels like entering a game lobby UI.

## Features
- Black/charcoal/off-white palette
- HUD-style navigation and panels
- Full-screen hero with typed animation
- Sections: Start, Profile, Loadout, Achievements, Invite
- Smooth scroll + section activation
- Canvas grid + particles background
- Subtle scanlines, glow, and transitions

## Getting started
1. Open `index.html` in a browser.
2. The site is static and requires no build.

## Edit
- `index.html` - structure and content
- `styles.css` - all theming and interactions
- `script.js` - animations, navigation and particle background

## Notes
- Maintain the non-color constraint: avoid adding bright hues.
- To customize text, edit the hero tagline and section copy in `index.html`.
